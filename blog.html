Tips on Web Development

padding/margin

we usually use a lot of padding when we make websites for desktops, to make them more attractive. While making it responsive for mobiles, tablets try decreasing the existing padding and margins

Use of em/rem/percentage

alwase try using em/rem/percentage instead of px, so that the text, image size adjust with respect to the device width

Flex-wrap

using flexbox to align HTML elements such as "div" ,"p" etc provides the force elements onto one line or can wrap onto multiple lines according to their width

Media query

media queries should be used to set width and height according to the breakpoints. Breakpoint refers to the width at which the website starts looking distorted

Box-sizing

It resolves a lot of problems padding causes, using box-sizing on HTML elements with a percentage width will take padding into account rather than having to adjust the width due to padding.

Some useful Extensions for various Softwares like Google Chrome, VScode

Google Chrome

  • Grepper

    It is the ultimate RAM upgrade for software developers brain. Easly snag code examples from around the web, then access your code examples without having to worry

  • Extensity

    this lets you manage your extensions. you can turn on/off the extension with just a click. It also allows you to always enable only one extension and turning off all others

  • Fake filler

    It is a form filler to fill all input fields on a page with randomly generated fake data. This is a productivity booster for developers and testers who work with forms as it eliminate manual entry in the form

  • GitHub code folding

    It gives you the ability to selectively hide and display sections of the code, it is already an invaluable feature in many IDEs and text editors. now developers can use the same code folding while poring over source code on github. works for any type of indentation, spaces or tabs.

  • Dimensions

    this extension measures the dimensions from your mouse pointer up/down and left/right until it hits a border. so if you want to measure distance between elements on a websites. it doesn't really work with images because there the colors change a lot pixel to pixel

  • Hide Scrollbar

    sometimes you hate the scrollbar on some sites. thats where this extension comes into play. this allows you to hide scrollbar on any page while still allowing you to use the scroll functionality. IT PROVIDE A MINIMAL STYLE FOR ANY WEBPAGE
